## Задача 1. BPMN-модель «Выдача IT-оборудования»

Схема в нотации BPMN 2.0, описывающая процесс выдачи IT-оборудования пользователю.

![BPMN-схема](bpmn-it-equipment.svg)

### Аннотации к схеме (противоречия и пробелы в описании)

1. **Пользователь:** Не указаны сроки обработки заявки и доставки оборудования (SLA).
2. **Первая линия ТП:** Отсутствует проверка возможности выдачи запрашиваемого оборудования данному пользователю (нет сценария отказа).
3. **Отдел снабжения:** Не описана проверка оборудования от поставщика на соответствие заявке.

## Задача 2. User Story + Use Case «Публикация товара на витрине маркетплейса»

### User Story

> Я как продавец, хочу опубликовать карточку своего товара на витрине маркетплейса, чтобы покупатели могли его найти и приобрести, а я — получать продажи.

### Use Case: «Опубликовать товар на витрине маркетплейса»

| Раздел | Описание |
|:---|:---|
| **Название** | Опубликовать товар на витрине маркетплейса |
| **Акторы** | Продавец, Модератор |
| **Предусловие** | Продавец авторизован в личном кабинете |
| **Основной сценарий** | 1. Продавец переходит в раздел «Мои товары».<br>2. Нажимает кнопку «Добавить товар».<br>3. Заполняет карточку товара: название, категория, цвет, тип, цена, описание, фото.<br>4. Нажимает «Отправить на модерацию».<br>5. Система сохраняет товар со статусом «На модерации».<br>6. Модератор проверяет товар на соответствие правилам.<br>7. Модератор одобряет товар → статус «Опубликован».<br>8. Продавец видит товар в разделе «Активные товары». |
| **Альтернативный А1: Незаполненные поля** | 4а. Система подсвечивает незаполненные обязательные поля.<br>4б. Продавец заполняет их и возвращается на шаг 4. |
| **Альтернативный А2: Модерация отклонена** | 7а. Модератор отклоняет товар с указанием причины.<br>7б. Товар получает статус «Отклонён».<br>7в. Продавец видит товар в разделе «Отклонённые» с причиной отказа.<br>7г. Продавец может отредактировать и повторно отправить на модерацию (возврат на шаг 3). |
| **Постусловие** | Товар опубликован на витрине маркетплейса и доступен покупателям для просмотра и покупки. |
